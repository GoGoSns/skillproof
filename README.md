# SkillProof

> On-chain provenance, attribution, and evolution tracking for Hermes Agent skills.

## Vision

A decentralized layer that lets Hermes skill authors prove ownership, 
track skill evolution across forks, and receive optional appreciation tips — 
all on-chain via Base Sepolia + IPFS.

## Why?

The Hermes Agent ecosystem already has 30+ community skills — but no way to:
- Prove who originally wrote a skill (attribution problem)
- Track how skills evolve when forked or self-improved
- Reward authors when their skills are used

SkillProof solves all three.

## Stack

- **Smart Contract**: Solidity on Base Sepolia
- **Storage**: IPFS via Pinata
- **Agent Integration**: Hermes Python skills
- **Dashboard**: Next.js + viem
- **Tip Layer (bonus)**: x402 over Base Sepolia
