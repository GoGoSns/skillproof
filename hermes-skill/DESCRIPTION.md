# SkillProof - Web3 Provenance for Agent Skills

This skill pack adds on-chain provenance, attribution, and evolution tracking to Hermes Agent skills.

## What's inside

- **skillproof/** — Register, verify, and prove ownership of skills on Base Sepolia + IPFS

## Why?

The Hermes Agent ecosystem has 30+ community skills, but no way to:
- Prove who originally wrote a skill
- Track how skills evolve when forked
- Reward authors when their skills are used

SkillProof solves all three using:
- Solidity smart contract (Base Sepolia)
- IPFS via Pinata for content storage
- keccak256 content hashing for integrity
